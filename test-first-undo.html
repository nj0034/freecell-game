Test: First Move Undo

This test verifies that the undo button works correctly for the very first move in a game.

Test Steps:

  1. Start a new game
  2. Record the initial state (moves, score)
  3. Make ONE move (any valid move)
  4. Verify state changed (moves increased)
  5. Click Undo
  6. Verify state returned to initial (moves = 0)

Manual Test Instructions:

  1. Open the game at http://localhost:3000
  2. Start a new game (if needed)
  3. Note the move counter shows "0"
  4. Make ONE move - click any card to move it

  6. Click the Undo button
