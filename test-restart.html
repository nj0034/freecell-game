💡 새로운 기능

1. 다시 시작 (Restart Same Game)

  • 현재 게임과 동일한 카드 배치로 다시 시작
  • 게임을 처음부터 다시 플레이 가능
  • 게임이 시작되지 않았을 때는 비활성화

2. 새 게임 확인 모달

새 게임 시작
현재 게임을 포기하고 새 게임을 시작하시겠습니까?
  • 게임 진행 중 새 게임 클릭 시 확인 모달 표시
  • 실수로 게임을 잃는 것을 방지

🍔 햄버거 메뉴 내용

  • 🎯 Safe Mode 토글
  • 🎨 테마 선택

  • ~~힌트~~ (제거됨)
